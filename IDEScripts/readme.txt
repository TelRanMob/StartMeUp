Please, add to this folder IDe scripts.
Here is the template for Ide script name:
<TCnumber>_<JiraTaskNumber>_<yourname>.html
EXAMPLE: 1089_STAR3_Alex.html
Here is the template for test suite name:
Siute_<TCnumbers>_<JiraTaskNumber>_<yourname>.html
EXAMPLE: Siute_108910901091_STAR3_Alex.html
 If you create a test suite, create here a separate folder for it. Folder name must contain suit name.
