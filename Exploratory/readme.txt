Please, add to this folder Reports from exploratory testing.
Here is the template for report name:
<JiraTaskNumber>_<reportDate>_<yourname>.html

Report creation workflow:
1. Perform testing session with SessionTester. 
2. Generate Web report (action|generate web report)
3. From the folder with generated report take ONLY ONE file with your last test session. 
4. Review and rename it following template.
5. Put the report file into 'Exploratory' folder
